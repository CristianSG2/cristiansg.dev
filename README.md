# cristiansg.dev

Portfolio personal de [Cristian Sánchez Gutiérrez](https://cristiansg.dev) — Junior Web Developer especializado en PHP & Laravel.

## Stack

- **[Astro](https://astro.build)** — generación estática
- **SCSS** — estilos con paleta Catppuccin Mocha/Latte
- **Vercel** — despliegue continuo desde `main`

## Desarrollo local
```bash
npm install
npm run dev        # dev server en localhost:4321
npm run build      # build de producción
npm run preview    # preview del build
```

## Estructura
```
src/
├── components/    # Componentes Astro reutilizables
├── layouts/       # Layouts base
├── pages/         # Rutas (index.astro)
└── styles/        # SCSS global y variables
public/            # Assets estáticos
```
